# ScoreFlow

**ez pz golf tournament software**

Golf tournament management app voor golfclubs.

## Features
- Wedstrijdbeheer (9 spelvormen)
- Speler portaal (mobiel)
- Live leaderboard
- Groepswedstrijden
- Eclectic seizoen scoring
- Supabase database koppeling
